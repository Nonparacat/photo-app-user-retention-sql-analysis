# Excel Dashboard Guide｜Excel 看板制作说明

建议使用以下 CSV 文件制作 Excel 看板：

1. `data_sample/daily_metrics.csv`
2. `data_sample/retention_summary.csv`
3. `data_sample/funnel_summary.csv`
4. `data_sample/channel_summary.csv`

## 建议看板结构

### KPI 区域

- 总注册用户数
- 总活跃用户数
- 总收入
- 付费用户数
- 付费转化率
- ARPPU

### 趋势图

使用 `daily_metrics.csv` 制作 DAU、新增用户、收入趋势图。

### 留存分析

使用 `retention_summary.csv` 对比整体、小红书、抖音广告、App Store 等渠道的 D1 / D7 留存率。

### 转化漏斗

使用 `funnel_summary.csv` 制作注册 → 打开 → 上传 → 使用滤镜 → 导出 → 订阅点击 → 付费漏斗图。

### 渠道质量对比

使用 `channel_summary.csv` 比较注册用户数、激活率、付费转化率、收入和 ARPPU。
