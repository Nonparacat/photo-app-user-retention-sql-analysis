# GitHub Upload Steps

## 1. 新建仓库

Repository name:

```text
photo-app-user-retention-sql-analysis
```

Description:

```text
SQL and Excel-based user retention and conversion analysis project for a photo editing app.
```

选择 Public，不要勾选初始化 README。

## 2. 上传方式

1. 解压 zip；
2. 进入 `photo-app-user-retention-sql-analysis` 文件夹；
3. 打开 GitHub 新仓库页面；
4. 点击 `uploading an existing file`；
5. 从 Finder / 文件管理器中拖入整个项目文件夹内的所有内容；
6. 确认能看到 `data_sample/`、`sql/`、`src/`、`figures/` 等路径；
7. Commit message 写：`Initial SQL portfolio project`。
