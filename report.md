# 影像类 App 用户行为分析与留存转化项目报告

## 1. 项目背景

本项目模拟一个影像类 App 的用户行为数据，覆盖注册、打开 App、上传照片、使用滤镜、导出图片、分享图片、点击订阅和付费购买等核心路径。

项目目标是通过 SQL 和 Excel 分析用户活跃、留存、功能转化、付费转化和渠道质量，形成适合数据分析、经营分析和 BI 报表岗位展示的完整作品集。

## 2. 数据概况

- 注册用户数：5,000
- 行为事件数：34,482
- 订单数：175
- 分析时间范围：2026-05-01 至 2026-06-30

## 3. 分析指标

本项目重点分析：

- DAU、新增用户数、活跃中新用户占比；
- 次日留存率、7日留存率、不同渠道留存表现；
- 注册到付费的核心转化漏斗；
- 付费转化率、总收入、ARPPU、复购率；
- 功能使用行为与付费转化的关系。

## 4. 主要发现

### 4.1 渠道质量差异明显

小红书和 App Store 渠道通常具有更好的留存和付费转化表现，说明这类渠道用户需求更明确，和影像类 App 的使用场景匹配度更高。

抖音广告渠道注册规模较大，但留存和付费转化偏弱，说明投放带来的部分用户可能只是浅层兴趣用户，需要进一步优化广告素材、人群定向和落地页承接。

### 4.2 功能使用行为与付费转化相关

使用滤镜、导出图片的用户往往比仅打开 App 的用户更接近付费转化。  
这说明核心功能体验是影响会员转化的重要节点。

### 4.3 导出到订阅点击存在流失

在用户完成图片导出后，并不是所有用户都会点击订阅入口。  
该环节可以考虑增加会员权益提示，例如高清无水印导出、批量修图、高级滤镜包、AI 修图增强等。

## 5. 业务建议

1. 对小红书和 App Store 渠道增加预算权重，优先获取高质量用户；
2. 对抖音广告渠道重点分析注册后行为路径，优化投放素材和转化承接；
3. 在图片导出成功页增加会员权益提示，提高订阅点击率；
4. 对使用滤镜但未付费用户进行定向运营，例如限时会员试用或滤镜包优惠；
5. 对 iOS 用户测试更高客单价会员权益组合，提高收入效率。

## 6. 面试讲解重点

面试时可以重点说明：

- users、events、orders 三张表如何关联；
- DAU、留存率、转化率、ARPPU 的计算口径；
- 如何从 SQL 指标进一步形成业务建议；
- Excel 看板如何服务运营和经营决策。
